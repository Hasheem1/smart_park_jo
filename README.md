ğŸ” QR Code Automation System

Smart Park JO uses an advanced QR code system to fully automate parking operations:

After reserving a parking space, the driver receives a unique QR code.

At the parking entrance, the QR code is scanned to:

Start the parking time

Begin cost calculation

Upon exit, the QR code is scanned again to:

Stop the reservation

Generate the final bill

Automatically deduct the amount from the driver's wallet

Drivers must add funds to their in-app wallet before reserving, ensuring a fast, secure, and cash-free parking experience.
